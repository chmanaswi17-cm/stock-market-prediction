# Stock Market Prediction

This project predicts Apple stock prices using Machine Learning (Linear Regression).

Files:

* AAPL.csv
* preprocess.py
* train_model.py
* predict.py
* evaluation.py

Libraries:

* Pandas
* NumPy
* Matplotlib
* Scikit-Learn
* Joblib

How to Run:

1. python train_model.py
2. python predict.py
3. python evaluation.py
